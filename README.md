# WebChat
Making a web chat to train with Node.js &amp; Socket.io

## Requirements
- Git
- Node.js >= LTS 10.15.3

## Install
```bash
git clone https://github.com/Noxilex/WebChat
cd WebChat
npm install
```

## Start
```bash
npm start
```
- Open browser on : `localhost:5001`
